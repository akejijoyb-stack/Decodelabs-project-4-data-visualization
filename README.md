# Decodelabs-project-4-data-visualization
## Project Overview

This project focuses on creating an interactive E-Commerce Dashboard in Microsoft Excel to transform raw business data into meaningful visual insights. The dashboard was designed to communicate key business metrics clearly, support data-driven decision-making, and demonstrate effective data storytelling through visualization.

## Project Goal

To create visual representations of data that communicate insights clearly and effectively.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Dashboard Design Techniques

## Dashboard Components

### Key Performance Indicators (KPIs)

* Total Revenue: $1,264,761.96
* Total Orders: 1,200
* Average Order Value (AOV): $1,053.96
* Median Order Value (MOV): $823.60

### Visualizations

#### Revenue Trend Analysis

A line chart showing revenue performance across different periods to identify sales patterns and trends.

#### Referral Source Analysis

A bar chart comparing revenue generated from various acquisition channels.

#### Payment Method Analysis

A column chart displaying customer payment preferences.

#### Interactive Filters

Slicers were implemented for:

* Coupon Code
* Order Status
* Date

## Key Insights

* The business generated **$1.26M in revenue from 1,200 orders**, with Instagram and Email emerging as the strongest revenue drivers.

* The **Average Order Value ($1,053.96)** exceeds the **Median Order Value ($823.60)**, indicating the presence of high-value transactions that increased overall revenue.

* **Online payments** were the most preferred payment method, highlighting customers' preference for digital transactions.

* Revenue trends show fluctuations over time, presenting opportunities to investigate peak-performing periods and replicate successful strategies.

## Business Recommendations

1. Invest more in high-performing marketing channels such as Instagram and Email.
2. Analyze peak revenue periods to identify successful campaigns.
3. Strengthen referral programs to increase customer acquisition.
4. Continue promoting digital payment methods for customer convenience.
5. Monitor high-value orders separately to understand premium customer behavior.
